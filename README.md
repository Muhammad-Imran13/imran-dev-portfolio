# Modern Dark Landing Page

This repository contains a responsive dark-themed landing page developed for a web design assignment. It demonstrates the use of interactive UI elements using pure CSS.

## Key Features
- **Mobile Navigation:** Features a custom CSS-only slide-in menu for mobile devices.
- **Dark Aesthetic:** A professional dark theme with vibrant accent colors.
- **Interactive UI:** Smooth transitions and button animations.

## Technologies Used
- HTML5
- CSS3 (Animations & Mobile-first design)
